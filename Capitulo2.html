Exercicios de HTML

2.1-Qual é o elemento HTML que contém todos os demais elementos de uma página?

R:O elemento HTML que envolve todos os outros elementos de uma página é o "html". Ele abriga todo o conteúdo da página, incluindo o cabeçalho "head" e o corpo "body", além de outros elementos como títulos, parágrafos, listas, imagens, links e scripts.

2.2-Qual é o elemento HTML que contém elementos com metainformações sobre uma página web?

R:O elemento HTML que contém metainformações sobre uma página web é o head. Ele inclui elementos como título da página, metadados de SEO, definições de caracteres, vinculação de folhas de estilo e scripts. O head é essencial para fornecer informações importantes sobre o conteúdo da página aos mecanismos de busca e ao navegador.

2.3-Qual é o elemento HTML que contém todos os elementos de conteúdo de uma página web?

R:O elemento HTML que contém todos os elementos de conteúdo de uma página web é o "body". Ele engloba todo o conteúdo visível da página, como textos, imagens, links, listas, formulários, vídeos, entre outros elementos. O body é onde o conteúdo principal da página é colocado e exibido no navegador.

2.4-Qual é o elemento HTML que permite associar um documento CSS a uma página web na seção de metainformações?

R:O elemento HTML que permite associar um documento CSS a uma página web é o "link". Ele é usado na seção "head" da página, com o atributo "rel" definido como "stylesheet" e o atributo "href" especificando o caminho do arquivo CSS a ser vinculado.
